# 첫 번째 포스트 샘플

이 글은 GitHub Pages와 `Just the Docs` 테마가 정상적으로 연결되었는지 확인하기 위한 샘플 포스트입니다.

## 무엇을 적으면 좋을까

- 이 블로그를 왜 시작했는지
- 앞으로 어떤 주제를 기록할지
- 오늘 작업하면서 배운 점 하나

## 예시 문단

정리해두지 않으면 금방 잊어버리는 내용들을 짧게라도 남기는 공간으로 쓰기 좋습니다. 나중에 글 수가 늘어나면 카테고리 페이지나 소개 페이지를 추가해도 자연스럽게 확장할 수 있습니다.
